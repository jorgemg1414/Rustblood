# Rustblood Official Website

Sitio web oficial de **Rustblood**, banda de progressive death metal originaria de Tepatitlán de Morelos, Jalisco, México.

Formada en enero de 2024, Rustblood busca forjar nuevos caminos en la brutalidad sónica tradicional, fusionando la crudeza del death metal con elementos progresivos y pasajes melódicos.

**Integrantes:**
- Edson Muñoz - Guitarra y Voces
- Joshua Giacomo - Guitarra
- Lemmy Yeudhiel - Bajo
- Alan Gómez - Batería

El sitio cuenta con información sobre la banda, discografía, galería de fotos, próximos eventos y medios de contacto.
